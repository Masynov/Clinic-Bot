# Clinic Telegram Bot

An asynchronous Telegram bot designed to automate clinic workflows, handle patient inquiries, and manage local database records.

## 🚀 Features
- Automated greeting and interactive menu navigation.
- Patient data management stored securely in a local SQLite database.
- Asynchronous architecture for fast and efficient handling of user requests.

## 🛠 Tech Stack
- **Language:** Python 3
- **Framework:** Aiogram 3 (Asynchronous Telegram Bot API)
- **Database:** SQLite
